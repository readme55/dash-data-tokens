Dash Platform Sample Dapps

This is a collection of simple Dash Platform Dapps to showcase how they can be implemented by third-party developers. These samples use the "Simple Dapp Signing" algorithm implemented by Chrome Wallet. This is not an official solution!

Checkout the Dash Sample Dapps GitHub repository to learn how to develop your own Dapp using the same method then these samples e.g. note-dapp.js.

Instructions:
1. Activate the "Simple Dapp Signing" switch in Chrome Wallet
2. On this website Sign-In using the button in upper right corner, enter your registered Username (from Chrome Wallet) and press "Sign-in"
3. Receive a confirmation dialog from Chrome Wallet and confirm it. You should now be logged in and ready to start a Dapp.

Note

Simple Note Dapp as sample for saving documents on Dash Platform using your Dash Username. The deployed data-contract uses objects for date, title, text and encryption status. To use this dapp you need to "Sign in" and then start the Note-Dapp. Create the note, press "Submit" on the webpage and wait for Chrome Wallet dialog to appear. Then confirm the dialog message.

PayMe

PayMe is a simple example to submit a payment requested from a website to Chrome Wallet. It uses your Dash Username and you just have to "Confirm" the payment request in the Wallet after submission.
