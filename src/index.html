Dash Platform Data Tokens

This is a collection of Prototype Token Dapps implemented on Dash Platform. These samples use the Simple Dapp Signing algorithm implemented by Chrome Wallet.

Checkout the Dash Sample Dapps GitHub repository to learn how to develop your own Dapp using the same method then these samples e.g. note-dapp.js.

Instructions:
1. Activate Push Notifications switch in Chrome Wallet. If you want to create your own Token run in Open Tab (only needed for extension) mode (bottom right), then a newly created token contract id is displayed after creation
2. On this website Sign-In using the button in upper right corner, enter your registered username (from Chrome Wallet) and press "Sign-in"
3. Receive a confirmation dialog from Chrome Wallet and confirm it. You should now be logged in and ready to start the Token Manager Dapp.
4. Default Token Contract is Fun Token. Ask for some tokens on discord/twitter or send a transaction to username "readme" with amount 0 (zero transaction) and a message saying "Request Tokens".

Token Manager

Create Tokens using your identity id as account and using your username on the frontend. A new Data-Contract for your Token will be created during the process. You need to read the new contract id from Chrome Wallet and insert it into the "Token Contract" field. Next Mint initial supply for the new Token. Thats it!
Get Account Balance will fetch the amount of tokens on your account. You can Transfer Tokens to any other username and even attach a message. The Token Explorer will show transaction history of any other account when inserting his username.
